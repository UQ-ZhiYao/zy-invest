How Daily Fee Accrual Works

Base Fee (daily)
Base Fee = AUM × rate% / 365

Accrues every calendar day regardless of performance. Deducted from NTA daily.

Performance Fee (daily)
If daily_return > hurdle/365:
Perf Fee = AUM × excess × rate%

Only accrues on days the fund outperforms the daily hurdle rate. Zero on underperforming days.
